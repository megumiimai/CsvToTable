# CsvToTable

CSV または Excel を HTML TABLE に変換

データを入力または貼り付けして、「変換」ボタンを押してください。

区切り符号は自定義可能です。

HTMLの `<table>` タグを使ってテーブルデータへ変換します。

https://megumiimai.github.io/CsvToTable/
